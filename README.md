# 🚀 rocket-market
쿠팡을 모티브로 만든 이커머스 서비스 API 서버입니다.

## 🚩 프로젝트 목표
- **유지보수성과 확장성을 고려한 객체지향 설계**

- **대용량 트래픽을 처리할 수 있는 아키텍처 설계**

- **테스트 코드를 통한 신뢰성 향상**

- **CI/CD 파이프라인 구축**

- **API 문서화 자동화**
